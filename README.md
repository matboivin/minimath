# minimath

Mini linear algebra library for 42 School Graphics projects.

### Prerequisites

GNU make and a C compiler.

### Installing

Clone the repository, change it to your working directory and run:
```console
$ make
```

To use the library:
```console
$ gcc foo.c -L. -lminimath -lm
```
